# Bobomb Static

Эта папка содержит JSON файлы обновлений и бета-сервисы для Bobomb.

## 📁 Файлы

- `android_update.json` - Обновления Android приложения
- `cli_update.json` - Обновления CLI/TUI версии
- `services_update.json` - Обновления базы сервисов
- `services/` - Файлы сервисов
- `services/main/services.bsl` - Главный файл сервисов 
- `services/beta/services.bsl` - Главный файл сервисов (beta)

## 🌳 Ветка static

Внимание! Если вы отследили, что Bobomb бьется на этот репозиторий, он просто проверяет обновления сервисов и программ.
